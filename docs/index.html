krnl; a bare bones x86 compatible kernel

krnl: this marks the beginning of the development of krnl. i wish to learn about working with the bare metal hardware to create a kernel capable of booting into the system and work further from there. this is NOT a project that aims to solve a problem, rather it aims to mark my personal interest in low level program development and is NOT fit for production use.

what krnl is capable of:
  • booting into system main memory using GNU GRUB as the bootloader
  • start an init process which will load all the drivers
  • loading drivers for a keyboard, a mouse to allow for input
  • capable of mounting memory as a file system, and allowing for temporary storage of any data
  • further capabilities will be added .....
what krnl will NOT be capable of:
  • having a rendering library that allows for graphics to work
  • network capabilities
  • this list will also be extended as development goes on
